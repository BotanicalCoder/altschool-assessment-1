# Fluffy Banana Pancakes Recipe

A simple, clean HTML-based recipe page for making delicious fluffy banana pancakes.

## Project Overview

This project is a beginner-friendly recipe webpage that provides step-by-step instructions for preparing fluffy banana pancakes. It's designed as an easy breakfast recipe that requires minimal ingredients and cooking time.

## Project Structure

```
.
├── index.html              # Main recipe page with ingredients and instructions
├── banana-pancakes.svg     # Illustration of the finished pancakes
└── README.md              # This file
```

## Recipe Details

- **Prep Time:** 10 minutes
- **Cook Time:** 15 minutes
- **Servings:** 4 pancakes
- **Difficulty Level:** Easy

## Ingredients

- 2 ripe bananas, mashed
- 1 cup all-purpose flour
- 1 cup milk
- 1 large egg
- 1 tablespoon sugar
- 1 teaspoon baking powder
- 1/2 teaspoon vanilla extract
- Pinch of salt

## How to Use

1. Open `index.html` in any web browser
2. Follow the ingredients list and instructions provided
3. Enjoy your freshly made fluffy banana pancakes!

## Tips

- Use very ripe bananas for the sweetest, fluffiest pancakes
- For extra flavor, add a pinch of cinnamon or top with fresh berries and maple syrup
- Approximate nutrition facts: 250 calories, 8g protein, 35g carbohydrates, 8g fat per serving

## Technical Stack

- HTML5 semantic markup
- SVG image format for illustration
